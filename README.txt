ApexGrowth.pro Next.js Production Project

How to deploy on Vercel:
1. Unzip this folder.
2. Upload to GitHub as a new repository.
3. In Vercel, import the GitHub repo.
4. Framework preset: Next.js.
5. Click Deploy.
6. Your existing domain apexgrowth.pro can stay connected to the Vercel project.

For manual deployment:
- You can also drag the entire unzipped folder into Vercel.
- Vercel will detect Next.js automatically.
